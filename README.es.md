## ¿Qué es Markdown?

> Markdown es un lenguaje de marcado ligero que permite formatear texto usando símbolos simples.
> Se usa ampliamente en READMEs de GitHub, documentación, wikis, blogs y notas.

## Encabezados

Los encabezados ayudan a estructurar tu README en secciones claras y legibles.

```md
# Encabezado 1
## Encabezado 2
### Encabezado 3
```

## Formato de Texto

**Texto en negrita**
```md
**Texto en negrita** o __Texto en negrita__
```

*Texto en cursiva*
```md
*Texto en cursiva* o _Texto en cursiva_
```

***Negrita + Cursiva***
```md
***Negrita + Cursiva***
```

~~Tachado~~
```md
~~Tachado~~
```

`código en línea`
```md
`código en línea`
```

## Listas

```

### Lista Ordenada

### Lista de Tareas
- [x] Tarea completada
- [ ] Tarea pendiente

- [ ] Tarea pendiente
```
