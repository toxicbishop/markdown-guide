## Markdown क्या है?

> Markdown एक हल्की मार्कअप भाषा है जो आपको सरल प्रतीकों का उपयोग करके टेक्स्ट को फॉर्मेट करने देती है।
> इसका व्यापक रूप से GitHub READMEs, डॉक्यूमेंटेशन, विकी, ब्लॉग और नोट्स में उपयोग होता है।

## शीर्षक

शीर्षक आपके README को स्पष्ट, पढ़ने योग्य खंडों में व्यवस्थित करने में मदद करते हैं।

```md
# शीर्षक 1
## शीर्षक 2
### शीर्षक 3
```
