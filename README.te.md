## Markdown అంటే ఏమిటి?

> Markdown అనేది సాధారణ చిహ్నాలను ఉపయోగించి టెక్స్ట్‌ను ఫార్మాట్ చేయడానికి అనుమతించే తేలికపాటి మార్కప్ భాష.
> ఇది GitHub READMEs, డాక్యుమెంటేషన్, వికీలు, బ్లాగ్‌లు మరియు నోట్స్‌లో విస్తృతంగా ఉపయోగించబడుతుంది.

```md

## హెడ్డింగ్ 2
### హెడ్డింగ్ 3
```
