## 什么是 Markdown？

> Markdown 是一种轻量级标记语言，可以使用简单的符号来格式化文本。
> 它广泛用于 GitHub README、文档、Wiki、博客和笔记。

## 标题

标题有助于将 README 组织成清晰、易读的部分。

```md
# 一级标题
## 二级标题
### 三级标题
```

## 文本格式

**粗体文本**
```md
**粗体文本** 或 __粗体文本__
```

*斜体文本*
```md
*斜体文本* 或 _斜体文本_
```

***粗斜体***
```md
***粗斜体***
```

~~删除线~~
```md
~~删除线~~
```

`行内代码`
```md
`行内代码`
```
