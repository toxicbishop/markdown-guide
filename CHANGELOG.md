# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Issue templates for bug reports and feature requests
- Pull request template
- GitHub Pages support with Jekyll
- Translations: Chinese (simplified), Spanish
- SECURITY.md for vulnerability reporting
- Project logo and banner in assets

## [1.0.0] - 2026-03-02

### Added
- Initial release of Markdown Guide
- Comprehensive documentation covering:
  - Headings
  - Text formatting (bold, italic, strikethrough, etc.)
  - Lists (ordered, unordered, task lists, nested)
  - Code blocks with syntax highlighting
  - Diff code blocks
  - Mermaid diagrams
  - Tables with alignment options
  - Blockquotes
  - Links and relative links
  - Images and image positioning
  - Video embedding techniques
  - GitHub Alerts (NOTE, TIP, IMPORTANT, WARNING, CAUTION)
  - Emojis shortcodes
  - Footnotes
  - Math expressions (LaTeX)
  - Escape characters
  - Badges (shields.io)
  - GitHub-specific features (autolinks, mentions)
  - HTML in Markdown
  - Collapsible sections
  - Anchors and table of contents
  - Keyboard keys styling
- README templates for different project types
- Quick reference card
- Useful tips section
- CONTRIBUTING.md with contribution guidelines
- MIT License
